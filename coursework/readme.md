## Overview

This curriculum synthesizes three elements:
1. **Rigorous physics sequence** (NGSS/AP aligned, algebra-to-calculus progression)
2. **SpaceX engineering case studies** (real-world motivation for every concept)
3. **Husmann Decomposition framework** (frontier research in Year 3)

Students don't just learn physics — they learn to *think like physicists building the future*.

## Key Equations Summary

### Classical Mechanics
```
F = ma                    Newton's Second Law
v² = v₀² + 2aΔx          Kinematics
p = mv                    Momentum
W = Fd cos θ              Work
E = KE + PE               Energy conservation
v² = GM(2/r - 1/a)       Vis-viva equation
```

### Electromagnetism
```
F = kq₁q₂/r²             Coulomb's Law
V = IR                    Ohm's Law
P = IV                    Power
F = qvB                   Lorentz force
EMF = -ΔΦ/Δt             Faraday's Law
```

### Quantum Mechanics
```
E = hf                    Photon energy
ΔxΔp ≥ ℏ/2               Uncertainty principle
Ĥψ = Eψ                   Schrödinger equation
```

### φ-Framework
```
φ = (1 + √5)/2 = 1.618...              Golden ratio
1/φ + 1/φ³ + 1/φ⁴ = 1                  Unity equation
α = 1/(N × W) ≈ 1/137                  Fine structure
n = log_φ(L/L_Planck)                  Bracket position
```

## Classroom Norms

1. **Evidence over authority** — Data matters, not credentials
2. **Precise language** — "It depends" is the start, not the end
3. **Wrong answers are data** — Failed predictions are the most interesting
4. **Frontier science is tentative** — "We don't know yet" is acceptable
5. **Critique reasoning, not people** — "That argument has a gap" not "you're wrong"
6. **Curiosity is mandatory** — The only unwelcome question mocks someone else
