## 5. The Look-Elsewhere Problem — Monte Carlo Results

### 5.1 The Raw Hit Count Test: FAILED (p = 0.45)

A Monte Carlo test of 100,000 random frameworks — each with random spectral constants drawn from uniform distributions — was run against the same 15 particle physics targets using the same search methodology (products of ≤ 3 building blocks from ~50 constants, giving ~26,000 combinations per framework).

**Result: the raw hit count is NOT significant.** The median random framework matches 13/15 targets at < 0.1%. The real framework matches 14/15. p = 0.45 — no signal whatsoever.

**Why:** With 26,234 product combinations and only 15 targets, the probability of at least one combination matching any given target at 0.1% is approximately 1 − (1 − 0.002)^26234 ≈ 1.0. The search space guarantees hits by combinatorial coverage alone. This is exactly the "36,000 combinations" problem that killed 57 out of 68 matches in the Theorem paper's original atomic scan.

**The 18 sub-0.1% matches from §2 are therefore NOT statistically significant as a group.** Any framework with ~50 building blocks will produce similar hit counts. We report this result honestly and without hedging.

### 5.2 Error Quality: Marginal (p = 0.074)

The geometric mean error of the real framework's matches (0.0013%) beats 90% of random frameworks with the same hit count. p = 0.074 — suggestive that the real framework produces *tighter* matches than random, but not decisive by any standard threshold.

## 6. What This Does NOT Explain

We present these results with full honesty about their limitations:

**No derivation of absolute masses.** The framework predicts mass RATIOS, not absolute values. Why the electron weighs 0.511 MeV rather than 5.11 MeV is not addressed. This is analogous to the atomic radius framework predicting vdW/cov RATIOS but needing the Bohr radius to set the scale.

**No mechanism for quark confinement.** The statement "the top quark sits 136 channels above the charm quark" is descriptive, not mechanistic. The framework does not derive QCD or explain why quarks confine into hadrons.

**No derivation of the Aufbau principle for particles.** Just as the atomic framework uses empirical electron configurations, the particle framework uses empirical masses. A true derivation would start from the spectrum and predict which eigenstates become fermions with which masses.

**Products of three constants offer combinatorial flexibility.** With 46 building blocks and products of up to 3, there are ~26,000 combinations. The raw hit-count Monte Carlo confirmed this concern at p = 0.45. The cross-domain coherence test (p < 10⁻⁵) survives because it tests SHARED constants, not independent hits — but the combinatorial caveat applies to any individual match taken in isolation.
